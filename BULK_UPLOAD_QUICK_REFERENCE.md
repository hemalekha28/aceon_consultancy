# 🚀 Bulk Upload - Quick Reference Card

## 3 Easy Steps

### Step 1: 📸 Select Images
- Click **"Bulk Upload"** in Product Management
- Select multiple images from your computer
- See thumbnails appear below

### Step 2: 📝 Fill Details
For each product:
- **Name**: Product title (e.g., "Memory Foam Mattress Queen")
- **Price**: In ₹ (e.g., 24999)
- **Stock**: Number available (e.g., 15)
- **Material**: Choose from dropdown (Latex, Coir, Memory Foam, Softy Foam, Spring)
- **Description**: Features and benefits
- **Size**: Optional (e.g., "Queen", "6x6")

### Step 3: ✅ Upload
- Click **"Upload X Products"** button
- See confirmation message
- Products appear in catalog immediately

---

## ⚠️ Important Rules

| What | Rule | Example |
|------|------|---------|
| **Files** | 50 max, 5MB each | JPG, PNG, GIF, WebP |
| **Product Name** | Required, max 100 | "Premium Latex Mattress 6x6" |
| **Price** | Required, > 0 | 24999 or 24999.99 |
| **Stock** | Required, ≥ 0 | 10, 50, 100 |
| **Description** | Required, max 2000 | "100% natural latex..." |
| **Material** | Required, select from list | Must pick one |
| **Category** | In database | latex, coir, memory-foam, softy-foam, spring |

---

## 🎯 Tips

✅ **Do This:**
- Write clear product descriptions
- Use good quality images (800x800px or larger)
- Double-check prices before uploading
- Fill all required fields (marked with *)

❌ **Don't Do This:**
- Use small/blurry images
- Leave required fields empty
- Use external image URLs
- Name products generically ("Product 1", etc.)

---

## ❓ Common Questions

**Q: Can I use external link images?**
A: No, upload images from your device only.

**Q: What if I make a mistake?**
A: Go to Product Management, click Edit (✏️), fix it.

**Q: How many products can I upload at once?**
A: 50 images max, but up to 100 products per batch.

**Q: Will my old products be affected?**
A: No, only adds new products.

## 🆘 If Something Goes Wrong

| Problem | Solution |
|---------|----------|
| Upload fails | Check file size (< 5MB) and format (JPG/PNG/GIF/WebP) |
| "Required field" error | Red highlighted field needs data |
| Invalid category | Select from dropdown, don't type |
| Image not showing | Use filename provided, not URL |
| Products not appearing | Refresh page after upload |

---

## 💡 Pro Tips

1. **Batch similar products** - Upload all mattresses first, then pillows
2. **Use consistent naming** - Helps customers find products
3. **Be descriptive** - More details = more sales
4. **Check stock counts** - Prevents overselling
5. **Review descriptions** - Fix typos before uploading
